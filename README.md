## Usage
Install `Node 10`
```
 git clone https://github.com/status-im/multisig-dapp.git
 cd multisig-dapp
 npm install
```

### Run on localhost:8000
```
 npm start
```

### Build Dapp into dist/
```
 npm run build
```

### Deploy Dapp into Decentralized Storage
```
 npm run deploy
```
