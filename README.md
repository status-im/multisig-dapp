# Multisig Dapp

Basic Dapp for deploying and using MultisigWallet.

## Changelog

v0.0.3:
- better routes
- better look'n feel 
- responds to tx receipts and errors
- allows deploy multisig wallet

v0.0.2: 
- bootstrap ui
- Identify if account is owner
- submit transaction
- list transactions
- confirm/revoke transaction 
- list owners
- remove owner
- add owner
- routes
